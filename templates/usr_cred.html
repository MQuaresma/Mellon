MellonFS - Login





Usage

MellonFS is a filesystem implemented in libfuse with enhanced authentication mechanisms. Each time an open system call is invoked, a temporary code will be sent to your designated email which you should insert in the Authentication Code menu.

Existing Users

To start the daemon insert your Username and press "Login".

New Users

To register as a new user fill in the Username and Email fields and insert the Master Key. By pressing the "Login" button the daemon will be started automatically.
